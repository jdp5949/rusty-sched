What rusty-sched offers beyond either

  • Single Rust binary — no Java runtime, no Node.js install, no Python venv.
  • Full Postgres backend via sqlx::Any — drop-in for HA / shared state.
  • JIL → REST translation — keep existing Autosys job definitions, manage them via a modern API.
  • RBAC + API keys built-in — Cronicle's API key UX, AutoSys's permission model, neither requires a plugin.
  • Audit log on every mutation, queryable via REST + UI.
  • MIT license, OSS.
